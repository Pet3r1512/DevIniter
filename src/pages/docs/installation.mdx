# Installation

Follow these following steps in order to install templates from DevIniter:

## Prerequisites

Make sure you have the following installed on your machine:

- [Node.js](https://nodejs.org/) (version 20)
- [PNPM](https://pnpm.io/) (version 9)

Note: We are working on to increase the number of Nodejs and PNPM version along with other package manager such as: NPM and Yarn.
