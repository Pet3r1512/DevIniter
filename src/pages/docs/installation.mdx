# Installation

To get started with DevIniter templates, please follow these steps:

## Prerequisites

Ensure you have the following tools installed on your system:

- <HighlightedText content={{href: "https://nodejs.org/", text: "Nodejs", target: "_blank" }} /> (v20.0.0 or higher)
- <HighlightedText content={{href: "https://pnpm.io/", text: "pnpm", target: "_blank" }} /> (v9.0.0 or higher)
- <HighlightedText content={{href: "https://www.npmjs.com/", text: "npm", target: "_blank" }} /> (v10.0.0 or higher)

<strong>Note:</strong> We are actively working to expand support for additional Node.js versions and package managers such as Yarn.

## Installing Template

We recommend starting a new template using `deviniter`, which downloads template for you automatically. To install a template, run:

<CodeBlockWithPkg npmCommand="npx @deviniter/cli@latest" pnpmCommand="pnpm dlx @deviniter/cli@latest" />

After running this command, our CLI will guide you through the setup process:

<QuestionBlock questions={[
    {
        question: "What is your project name?",
        description: "Enter a name for your new project (lowercase, no spaces)"
    },
    {
        question: "Select a template: (use arrow keys to navigate)",
        options: [
            "Next.js",
            "Vite",
        ]
    },
]} />

Since version <strong>1.2.14</strong>, DevIniter provides an option to decide whether to install Prisma when the <strong>Next.js</strong> template is selected:

<QuestionBlock questions={[
    {
        question: "Would you prefer to install Prisma? ",
        options: ["Yes", "No"]
    },
]} />

Once you've selected your template, DevIniter will automatically download and install all necessary dependencies for your project. This process may take a few minutes depending on your internet connection.

## Starting Your Project

Navigate to your project directory and start it locally using one of these commands:

<CodeBlock lang="bash" code="cd your-project" />

<CodeBlockWithPkg npmCommand="npm run dev" pnpmCommand="pnpm dev" />

If there is no other application running on port 3000, your project will start at <HighlightedText content={{href: "http://localhost:3000", text: "localhost:3000", target: "_blank"}} />.

<TipsBlock tips={[
    <p>- With <span className="text-[#e4c1f9] font-semibold">Vite</span> template, your project will start at <HighlightedText content={{href: "http://localhost:5173", text: "localhost:5173", target: "_blank"}} /> in default</p>
]} />

## Troubleshooting

### Common Issues

1. **Port Already in Use**
```bash
Error: Something is already running on port 3000
```
**Solution**: Either close the application using port 3000 or modify the port in your configuration file.

2. **Node Version Mismatch**
```bash
Error: The engine "node" is incompatible with this module
```
**Solution**: Use nvm (Node Version Manager) to install and switch to the required Node.js version:
<ScriptBlock script={
    {command: "nvm",
    content: "use 20"}
}/>

3. **Package Installation Fails**
- Clear your package manager cache:

<CodeBlockWithPkg npmCommand="npm cache clean --force" pnpmCommand="pnpm store prune" />

- Ensure you have the necessary permissions
- Check your network connection
