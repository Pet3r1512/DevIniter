# Installation

To get started with DevIniter templates, please follow these steps:

## Prerequisites

Ensure you have the following tools installed on your system:

- <Link href="https://nodejs.org/" className="text-primary font-semibold underline decoration-solid">Nodejs</Link> (version 20)
- <Link href="https://pnpm.io/" className="text-primary font-semibold underline decoration-solid">pnpm</Link> (version 9)
- <Link href="https://www.npmjs.com/" className="text-primary font-semibold underline decoration-solid">npm</Link> (version 10)

<strong>Note:</strong> We are actively working to expand support for additional Node.js, pnpm and npm versions, as well as other package managers such as Yarn.

## Initializing Your Project with DevIniter Templates

To install one of our templates, run the following command:

## Starting Your Project

After successfully installing your selected template, navigate to the project directory by running:

<ScriptBlock script={
    {command: "cd",
    content: "dev-app"}
}/>

Then, start the project locally with one of the following commands:

<InstallBlock command={
    [
        {
            value: "pnpm",
            script: "pnpm dev"
        },
        {
            value: "npm",
            script: "npm run dev"
        },
    ]
}/>

If there is no other application running on port 3000, your project will start at <Link href="http://localhost:3000" className="text-primary font-semibold underline decoration-solid">localhost:3000</Link>.
